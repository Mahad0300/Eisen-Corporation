# Eisen Corporation — Premium Car Dealership

Modern, responsive landing page inspired by classic auto export dealership layouts.

## Run locally

Open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Structure

- `index.html` — page markup
- `css/global.css` — design system (variables, components, responsive)
- `js/main.js` — slider, mobile nav, footer year

## Stack

Static HTML/CSS/JS — no build step required.
